
# Архитектура и Документация Проекта (Project Architecture)

## Описание проекта
Этот документ создан для того, чтобы даже через 10 лет любой разработчик (или вы сами) смог открыть этот проект и четко понять, что здесь происходит, как он устроен, зачем создавался и как работает.

Проект представляет собой небольшое, но высокотехнологичное веб-приложение, демонстрирующее передовые современные подходы к веб-дизайну (Glassmorphism, плавные микро-взаимодействия) и включающее развлекательный элемент — мини-игру "Поймай сферу" в центре экрана.

## История и Цель
- **Зачем (Why):** Изначально проект был запрошен как пример абсолютно надежного, проверенного временем стека технологий с ультрасовременным интерфейсом. Цель — показать, что веб-сайт может быть не только стабильным, но и эстетически безупречным, премиальным (используя современные шрифты, кириллицу, градиенты).
- **Для чего (What for):** Основное назначение — демонстрация "вау-эффекта" (wow-effect) в коротком веб-формате. В центре внимания — пользовательский опыт и развлечение через мини-игру. 

## Технологический стек (Что используется и почему)
Проект построен на надежных и проверенных решениях:
1. **React (через Vite) + TypeScript:** Строгая типизация гарантирует работу без ошибок (багов). React — индустриальный стандарт, а Vite обеспечивает сверхбыструю сборку.
2. **Framer Motion:** Лучшая современная библиотека для создания высокопроизводительных, физически корректных и плавных анимаций (spring animations, layout transitions).
3. **Люцидные Иконки (Lucide React):** Красивые, консистентные векторные иконки.
4. **CSS (Vanilla):** Мы намеренно используем классический CSS с мощью CSS-переменных, избегая перегруженности фреймворками, чтобы добиться максимального контроля над цветами, тенями и эффектами стекла (glassmorphism). Используется шрифт "Inter", который великолепно смотрится с кириллицей.

## Структура директорий
- `/src/main.tsx` — Точка входа в приложение, инициализация React дерева.
- `/src/index.css` — Глобальные стили, дизайн-система. Содержит CSS-переменные для градиентов, неоновых эффектов и утилит-классы для glassmorphism.
- `/src/App.tsx` — Главный компонент-контейнер. Отвечает за макет страницы и фоновые декорации.
- `/src/components/` — Директория для переиспользуемых модулей.
  - `Game.tsx` — Вся логика и UI мини-игры. Включает в себя таймер, систему очков и логику перемещения цели по экрану в зависимости от действий пользователя.

## Логика работы (Как это работает)
1. **Рендеринг:** При загрузке страницы `App.tsx` рендерит роскошный анимированный фон с движущимися градиентными сферами заднего плана.
2. **Взаимодействие:** В центре экрана располагается "Стеклянная карточка" (Glassmorphism card).
3. **Игровой цикл (Game Loop):** Внутри `Game.tsx` реализована логика мини-игры. 
   - Пользователь нажимает на кнопку "Начать". 
   - Запускается счетчик времени (например, 30 секунд).
   - Появляется интерактивная цель, которая меняет свою позицию случайным образом (внутри определенных границ), тестируя реакцию игрока.
   - За каждое успешное попадание начисляются очки.
   - Используются анимации Framer Motion, чтобы цель двигалась органично и живо.
4. **Завершение:** По истечении времени пользователю показывается красивый экран с результатами, и он может начать заново.

## Инструкция по запуску
Откройте директорию в терминале и выполните:
```bash
npm install
npm run dev
```
